Question 15

List and explain the services provided by OS for the user and efficient operation of system. (06 Marks)

The specific services provided, of course, differ from one operating system to another, but we can identify common classes. These operating-system services are provided for the convenience of the programmer, to make the programming task easier.

15.1 User Interface

Almost all OS have a user-interface (UI).

Different interfaces are:

15.1.1 CLI (Command Line Interface)

This uses

i. text commands and

ii. method for entering the text commands.

15.1.2 Batch Interface

Commands & directives to control those commands are entered into files, and those files are executed.

15.1.3 GUI (Graphical User Interface)

The interface is a window-system with a pointing-device to

i. direct I/0

ii. choose from menus and

iii. make selections.

15.2 Program Execution

The system must be able to

i. load a program into memory and

ii. run the program.

The program must be able to end its execution, either normally or abnormally.

15.3 I/O Operations

i. The OS must provide a means to do I/O operations because users cannot control I/O devices directly.

ii. For specific devices, special functions may be desired (ex: to blank CRT screen).

15.4 File-System Manipulation

i. Programs need to

ii. read & write files (or directories)

iii. create & delete files

iv. search for a given file and

v. allow or deny access to files.

15.5 Communications

In some situations, one process needs to communicate with another process.

Communications may be implemented via

i. Shared memory or

ii. Message passing

In message passing, packets of information are moved between processes by OS.

15.6 Error Detection

Errors may occur in

i. CPU & memory-hardware (ex: power failure)

ii. I/O devices (ex: lack of paper in the printer) and

iii. user program (ex: arithmetic overflow)

For each type of error, OS should take appropriate action to ensure correct & consistent computing.
